# Audio para la Boda

Para agregar música de fondo a la invitación:

1. Coloca un archivo de audio en formato MP3 en esta carpeta
2. Nómbralo `wedding-music.mp3`
3. El archivo se reproducirá automáticamente cuando el usuario active la música

**Nota:** Asegúrate de tener los derechos de uso de la música que elijas.

## Formatos soportados
- MP3 (recomendado)
- OGG
- WAV

## Recomendaciones
- Usa archivos de tamaño moderado para una carga rápida
- Considera usar música instrumental o suave
- El audio se reproducirá en loop (repetirá automáticamente)
